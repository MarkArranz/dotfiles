# dotfiles

Config files for the tools I like to use.

## File Locations 

Where to put each file on the filesystem.

| Tool Name | Config File     | Path                  | Rename to...  |
|:-:        |:-:              |:-:                    |:-:            |
| zsh       | zshrc           | $USER_HOME (~)        | .zshrc        |
| Alacritty | alacritty.yml   | ~/.config/alacritty/  | -             |
| tmux      | tmux.conf       | $USER_HOME (~)        | .tmux.conf    |
| LunarVim  | lvim_config.lua | ~/.config/lvim/       | config.lua    |
